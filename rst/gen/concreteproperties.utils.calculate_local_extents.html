concreteproperties.utils.calculate_local_extents#

calculate_local_extents(geometry: CompoundGeometry, cx: float, cy: float, theta: float) → Tuple[float][source]#

Calculates the local extents of a geometry given a centroid and axis angle.

Parameters

  • geometry (sectionproperties.pre.geometry.CompoundGeometry) – Geometry over which to calculate extents

  • cx (float) – x-location of the centroid

  • cy (float) – y-location of the centroid

  • theta (float) – Angle (in radians) the neutral axis makes with the horizontal axis (\(-\pi \leq \theta \leq \pi\))

Returns

Local extents (x11_max, x11_min, y22_max, y22_min)

Return type

Tuple[float]
